Detailed Description

Math operations: distance, trilateration and extrapolation.

Contains static methods get_distance() to get distance to a beacon, and methods TrilatLocation2d() and TrilatLocation3d() for the 2D and 3D trilateration respectively. Also inxtrapolate() for extrapolation.

Trilateration in 2D.

Trilateration in 2D with a minimum of reference 3 points (beacons). As the 3 or more distances cannot be satisfied exctly in general, the result is the least squares fit.

Used in Sensors::BeaconSensor.

Based on https://github.com/Wayne82/Trilateration
