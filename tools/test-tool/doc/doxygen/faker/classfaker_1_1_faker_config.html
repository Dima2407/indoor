Read config from a JSON file, returns true if success.

File example { "inBeaconsFile": "in_beacons.json", "inTimesFile": "in_times.dat", "outEventsFile": "out_measure.dat", "t1": 0.0, "point1": [ 0.2, 0.2, 0.0], "t2": 1.0, "point2": [0.8, 0.3, 0.0], "ticks" : 1000 } 

       @param fileName   File name
       @return           true is success
