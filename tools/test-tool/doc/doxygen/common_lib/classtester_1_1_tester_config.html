Read Tester config from JSON.

File example { "inBeaconsFile" : "in_beacons.json", "inEventsFile" : "out_measure.json", "point1": [ 0.2, 0.2, 0.0], "point2": [0.8, 0.3, 0.0], "ticks" : 1000, "skipEvents" : 4 } 

       @param fileName   The JSON file name
       @param verbose    true = print log to stdout

       @return           true if success
