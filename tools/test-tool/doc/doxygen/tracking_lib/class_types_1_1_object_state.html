Detailed Description

Object (Smartphone) state as a vector + matrix.

Contains a 8-vector of Object position, velocity, acceleration etc. in 2D only, and a 8x8 matrix (why the matrix). Also contains timestamp. This is a data class which basically has only data, getters and setters.

Uses Eigen::Matrix from Eigen Library.

Used by Sensors::AbstractSensor and other classes.

WARNING !!! Do not confuse this with ObjectState in common_types.h !
