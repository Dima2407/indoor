Tracking library description.

Author
Alex Pitertsev
Version
1.0
Date
November 2016

tracking_lib - library for the tracking of the user indoor movement. It uses bluetooth beacons and built-in android sensors like accelerometer, gyroscope and orientation sensor. Includes subfolder external and list of cpp and header files.
